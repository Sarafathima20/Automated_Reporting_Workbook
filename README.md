# Automated Reporting Workbook – Excel Dashboard

## Project Overview

This project focuses on building an automated reporting workbook using Microsoft Excel to transform raw sales data into meaningful business insights.

The workbook automatically updates reports and visual summaries using formulas, Pivot Tables, charts, and slicers to support efficient reporting and decision-making.

---

## Objective

The objective of this project is to:

* Automate sales reporting processes
* Generate dynamic business summaries
* Create interactive dashboards
* Reduce manual reporting effort
* Improve data visibility through visual analysis

---

## Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Excel Formulas
* Slicers
* Data Tables

---

## Workbook Structure

### 1. Raw_Data

Contains imported sales dataset used for analysis.

### 2. Dashboard

Interactive dashboard containing:

* Total Sales
* Total Orders
* Average Sales
* Total Profit

Visualizations:

* Sales by Category
* Sales by Region
* Sales by Ship Mode

Interactive Filters:

* Region
* Category
* Segment / Ship Mode

### 3. Automated_Report

Summary report generated using automated calculations including:

* Total Sales
* Total Orders
* Average Sales
* Total Profit
* Highest Sale
* Lowest Sale
* Last Updated Date

---

## Automation Features

* Dynamic formulas linked to source data
* Pivot Table reporting
* Interactive slicers
* Automatic report refresh when data changes
* Structured Excel Table for scalability

---

## Key Insights

* Sales performance can be monitored through interactive filtering.
* Regional trends provide business visibility.
* Shipping modes influence sales distribution.
* Automated summaries reduce manual reporting effort.

## Conclusion

This automated reporting workbook demonstrates how Excel can be used to build efficient and interactive business reporting solutions through automation and visualization techniques.
